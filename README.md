# 🚀 AI Automation Agents Portfolio

AI-powered automation systems built using **n8n, OpenAI, Pinecone, and multiple integrations**.
This repository showcases real-world AI agents, RAG systems, and automation workflows designed for business use cases.

---

## 🧠 Overview

This project is a collection of **production-ready AI agents and automation systems** that demonstrate:

* AI-powered decision making
* Retrieval-Augmented Generation (RAG)
* Multi-agent architectures
* Business workflow automation
* Multi-platform assistants (Telegram, WhatsApp, Email)

---

## 🔥 Key Projects

### 🤖 AI Assistants

* **Telegram AI Assistant (Advanced)**

  * Multi-modal (text, voice, image)
  * Memory (short + long-term)
  * Tool integration (email, calendar, search)

* **WhatsApp AI Assistant**

  * Conversational AI
  * Email sending via Gmail
  * Knowledge retrieval (Wikipedia)
  * Calculator tool

4. Start interacting with the AI agent
